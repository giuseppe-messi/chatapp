**ChatApp** is a modern real-time chat platform designed to demonstrate clean architecture, full-stack TypeScript practices, and production-ready setup for messaging features.

- Real-time messaging with **Socket.IO**
- Secure authentication with sessions and hashed passwords
- Type-safe backend with **Prisma ORM**
- Optimistic UI updates using **React Query**
- Responsive and modern UI powered by **Tailwind CSS**
- PostgreSQL for relational data, MongoDB for chat messages
- Clean separation between client and server
